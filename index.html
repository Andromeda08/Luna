Using Luna

If you only need to use the compiled CSS just download Luna and import it in your website's HTML.

<link rel='stylesheet' href='path/to/luna.min.css' />

If you want to work with Luna's source files clone the GitHub repository.

$ git clone https://github.com/Andromeda08/Luna
