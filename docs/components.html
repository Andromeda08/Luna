Avatars

For stlying avatars you have various choices.

Basic Avatar

Add .avatar to any <img> element to make it an avatar. By default avatars have rounded corners if you use .avatar-circle instead of .avatar you will get a circle shaped avatar.

Avatar Sizes

Avatars have a default size of 40px. Instead of adjusting the width and height attributes, you can also use the classes .avatar-[1-8]. The sizes go from 16px to 96px.

Avatar Groups

Currently supports up to 3 avatars. On hover the avatars are spread out if using the .ag-animated class.

Header

Luna provides multiple styles of headers that you can customize to your liking.

Customizing the header

The default header styles are Primary (the default), Light, Dark and Transparent, all of them refer to the headers background. Use these classes .navbar and.navbar-[light, dark, transparent] to apply your choice of style.

With the classes .navbar-container-[1-8] you can set the inline padding of the headers content.

Add the .navbar-logo class to a <p> or <a> to the element displaying your logo or website name.

It is your choice to include a right or left floating section in your header. The content of ul.navbar-nav floats to the left by default, add the .ml-a class to make it float right.

Add .navbar-animated to your <header> element to make its links have an underlined animation on hover.

You can add a bottom border to your header with the .navbar-border class.

Header Structuring

header.navbardiv.navbar-container-p[] div.navbar-collapseul.navbar-nav li.nav-itema.nav-link
