Introduction

Luna is a CSS front-end framework that is responsive and aims to be easily customizable using less.

Quick Start

Once you have downloaded Luna you can easily include it in your project with the following code.

<link rel='stylesheet' href='path/to/luna.min.css' />

Using Luna

You can use Luna as the base of your project.
When working with the source files of Luna you can freely customize it to your liking, expand on it, remove modules and so on.
Luna is written using the less css language extension similar to sass/scss.
To get started clone the Github repository with the following command:

$ git clone https://github.com/Andromeda08/Luna

Globals

These are important settings for Luna to work properly.

Doctype HTML5

<!doctype html>
<html>
...
</html>

Responsive Viewport
